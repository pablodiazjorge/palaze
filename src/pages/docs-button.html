Botón

<plz-button> | PlzButton

Los botones representan acciones accesibles desde un click por el usuario.

Se utilizan en distintos contextos y situaciones en una misma página. Pueden usarse para busquedas, para formularios, para contactos, para redes sociales, para cargas, etc.

Ejemplos

Variantes de botón

Usaremos el atributo variant para elegir el estilo de botón que se prefiera entre "default", "pop-shadow", "progress" y "icon".

Dentro de los usos del componente, los más usados son estas cuatro variantes.

Default Shadow Pop Link

Link en botón

Usaremos el atributo link para determinar la url a la que se accederá al hacer click en el botón. En el caso de no determinar una url mediante el atributo, se darña por entendido que se usa el botón con otro tipo de acción.

Para asegurarnos que el link se abre como queremos, haremos uso del atributo target, en este podremos elegir entre "_blank", "_self", "_parent", "_top" y "framename". En el caso de no hacer uso de este atributo, se utilizará por defecto "_self" (se abrirá la página en el mismo marco en el que se ha hecho el click).

Estos dos atributos se pueden usar con las distintas variantes de botón.

Link

Colores de botón

Usaremos el atributo color para elegir el color de botón que se prefiera entre "black", "purple", "blue", "green", "red", "orange", "yellow" y si no eliges ningún color tendrás el de por defecto.

Dependiendo del estilo de página en que se usen, es apropiado cambiar el color acorde a esta.

Variante Progress

Usaremos el atributo variant para elegir "progress". A diferencia de un botón normal, deberemos de hacer uso del atributo text para introducir información en el botón. También haremos uso del atributo active al que deberemos pasarle "true" o "false" dependiendo de si queremos que el botón esté activo o no. En el caso de que no esté activado, pasará a ser un botón normal hasta que el atributo active cambie. Este tipo de variente se suele usar principalmente en formularios, ya que puedes indicar al usuario que su petición se está procesando y una vez termine volverá a la normalidad (es dinámico).

Sobras de botón y variante PopShadow

Usaremos el atributo variant para elegir "pop-shadow" y el atributo shadow-color para elegir entre "black", "purple", "blue", "green", "red", "orange" y "yellow". También puedes elegir colores como hemos visto anteriormente con el atributo color.

Puedes buscar combinaciones entre los distintos atributos para dar forma, color, sombra y tamaño dependiendo de lo que estés buscando para tu sitio web.

Variante icono

Usaremos el atributo variant para elegir "icon" y el atributo icon para elegir entre "github", "linkedin" y "mail". Este tipo de botón se usa en gran variedad de ocasiones, pero siempre va a destacar para compartir información de redes sociales.

A continuación un ejemplo de los distintos iconos con diferentes tamaños. Recalcar que si deseas un icono distinto, siempre puedes acceder al fichero assets/icon e importar ahí una imagen con esta estructura "Icon-imagen.svg". La imagen será detectada como podemos observar en la segunda tanda de ejemplo.

Forma del botón

Usaremos el atributo corner para elegir la forma del botón que se prefiera entre "default", "low-rounded", "rounded", "square", y "circle".

Dependiendo del estilo de la página en el que se usen y los tokens de diseño, se utilizará una forma distinta.

Square Default Low Rounded Rounded Circle

Tamaño de botón

Puedes usar el atributo size para elegir el tamaño de botón que se prefiera entre "small", "default" y "big".

Dentro de estos tamaños, lo normal es uno ni grande ni pequeño, pero siempre hay casos donde puedes necesitar distintas opciones.

Atributos

Nombre Descripción Posibilidades Por defecto
