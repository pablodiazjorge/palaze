Tooltip

<plz-tooltip> | PlzTooltip

El componente tooltip consiste en un mensaje emergente que aparece en pantalla cuando el usuario pasa el cursor sobre un elemento concreto.

Puedes recorrer el cursor hasta

aquí,

Distintas posiciones para el tooltip

Usaremos el atributo position para elegir la posición del tooltip entre "top" y "bottom".

Como se puede observar, el componente tooltip puede adaptarse a distintos tipos de elementos, ya sean botones, icones, imagenes, textos, etc. Recalcar que lo más normal es utilizar palabras o frases concretas y resaltar información adicional interesante.

Distintos usos para el tooltip

En este ejemplo a continuación se observa que en el tooltip pueden aparecer otros elementos a parte de texto simple. Se puede introducir en el tooltip un elemento link para poder accder a más documentación o lo que consideres importante. También puedes dejar libre tu imaginación y añadir una pequeña separación en el texto para hacerlo más extenso, aunque no suele ser este el uso para el componente tooltip.
