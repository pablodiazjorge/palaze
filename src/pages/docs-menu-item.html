Menu-item

<plz-menu-item> | PlzMenuItem

Los componentes menu-item forman parte del componente padre Menu. Se utilizan como parte del menu para identificar las distintas secciones que se requieren.

Distintos tipos de menu-item

Usaremos el atributo type para elegir el tipo de menu-item que consideremos entre "default", "home", "settings", "user", "message".
