Section

<plz-section> | PlzSection

El componente section constará de varias opciones para estructurar secciones en una página web.

La función de este componente es meramente organizativa.

Ejemplos

Variante wrapper

Usaremos el atributo variant para elegir "wrapper".

Esta variente organiza elementos en columnas para distribuirlos equitativamente a lo largo del espacio.

Dependiendo de los elementos y de las dimensiones del dispositivo, el componente distribuirá los elementos contenidos en más filas.

Componente tarjeta

Se han añadido atributos para hacer aún más personalizable el componente.

Componente Separador

Se han añadido atributos para hacer posible aumentar el grosor del separador si se desea.

Componente Heroe

Se ha añadido una variante en la que se divide el contenido en dos columnas.

Variante columns

Usaremos el atributo variant para elegir "columns".

Esta variente organiza elementos en columnas al estilo de un artículo.

Dependiendo de los elementos y de las dimensiones del dispositivo, el componente distribuirá los elementos contenidos de columnas a filas.
