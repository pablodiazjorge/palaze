Atributos

Nombre Descripción Posibilidades
variant Distintas variantes del componente tarjeta 'default' - 'simple' - 'header' - 'footer' - 'image'
hover Distintos colores del componente botón 'gray' - 'blue' - 'green' - 'red'
image Dirección url de la imagen a colocar en las tarjetas con variante "default" y "image" './mi-imagen-ejemplo.svg'
url Dirección url para crear evento (solo para la variante "default") './ejemplo.html'
