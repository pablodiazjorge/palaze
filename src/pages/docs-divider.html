Separador

<plz-divider> | PlzDivider

Los separadores sirven para marcar distintas partes del contenido y hacer que se diferencien.

Ejemplos

Tipos de separador

Usaremos el atributo type para elegir el tipo de separador que veamos adecuado para nuestro contenido.

Tamaño del separador

Usaremos el atributo height para elegir el ancho del separador.

Color del separador

Usaremos el atributo color para elegir el color. Por defecto será black-4 (meter un tooltip aquí).

Puedes usar tanto hex, rgb o rgba para el atributo color.

Margen superior e inferior del separador

Usaremos el atributo space-top y space-bottom para añadir espacio entre los elementos y el separador.

El separador estará a 10px de este elemento.

El separador estará a 20px de este elemento.

Atributos

Nombre Descripción Posibilidades Por defecto
type Distintas tipos del componente separador 'solid' - 'dashed' - 'dotted' 'solid'
color Distintos colores del componente separador hex, rgb, rgba black-4
space-top Separación superior del componente separador Number '0'
space-bottom Separación inferior del componente separador Number '0'
