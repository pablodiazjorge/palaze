Separador

<plz-divider> | PlzDivider

Los separadores sirven para dividir distintas partes del contenido y hacer que se diferencien.

En este caso se hará uso de separadores de tipo horizontal ya que son los más comunes y se podrá ajustar grosor, color y forma del mismo.

El uso más normal de estos es de tipo minimalista, una línea fina solida de un color que vaya acorde al estilo de la página y que no resalte mucho.

Ejemplos

Tipos de separador

Usaremos el atributo type para elegir el tipo de separador que veamos adecuado para nuestro contenido entre "solid", "dashed" y "dotted".

En este caso estaremos modificando la forma del separador, pudiendo elegir entre tres tipos.

Grosor del separador

Usaremos el atributo height para elegir el grosor del separador.

Pese a que lo más común es utilizar un grosor pequeño, en algunas ocasiones puede verse el caso de necesitar un grosor más grande.

Color del separador

Usaremos el atributo color para elegir el color más adecuado según nuestro estilo de página web. Por defecto será black-4 (meter un tooltip aquí).

Se puede usar tanto hex, rgb o rgba para el atributo color.

Margen superior e inferior del separador

Usaremos el atributo space-top y space-bottom para añadir espacio entre los elementos y el separador.

En algunos casos es necesario añadir un margen extra para que la colocación de nuestro elemento separador se visualice correctamente con nuestro contenido.
