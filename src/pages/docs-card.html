Tarjeta

<plz-card> | PlzCard

Las tarjetas sirven para agrupar contenido.

Ejemplos

Variantes de la tarjeta

Puedes usar el atributo variant para elegir el estilo de botón que prefieras.

Default Progress Icon

Colores de botón

Puedes usar el atributo color para elegir el color de botón que prefieras.

Borde de botón

Puedes usar el atributo border para elegir la forma del botón que prefieras.

Tamaño de la tarjeta

Puedes usar el atributo size para elegir el tamaño de botón que prefieras.

Atributos

Nombre Descripción Posibilidades
variant Distintas variantes del componente botón 'default' - 'icon' - 'progress'
color Distintos colores del componente botón 'default' - 'blue' - 'green' - 'red'
size Distintos tamaños del componente botón 'default' - 'small' - 'big'
corners Distintas formas de las esquinas del componente botón 'default' - 'low-rounded' - 'rounded' - 'square' - 'circle'
icon Solo compatible con la variante "icon". Distintos iconos para la variante 'github' - 'linkedin' - 'mail' - 'palaze'
