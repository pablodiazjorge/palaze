Hero

<plz-hero> | PlzHero

El componenteherosirve para introducir un contenido de una forma atractiva. Se suelen usar en landing pages al inicio de la página.

En al gran mayoría de ocasiones se usa una imagen de fondo, un título y un subtítulo. En algunos casos, se usan botones o algún tipo de elementos con el que interactuar (a gusto el cliente).

Hero

Aquí suele ir una pequeña introducción al contenido de tu landing page, algo que capte la atención de los usuarios al visitar tu web.

Ejemplos

Heroe simple

Usaremos el atributo variant para elegir el tipo de heroe "simple".

En este caso, se opta por un componente básico, título y subtítulo. Debido a ser un componente, es personalizable y puedes variar según tus necesidades, se podría añadir tarjetas o un botón para acceder a otra sección, etc.

Heroe image

Usaremos el atributo variant para elegir el tipo de heroe "image".

En algunas ocasiones, solo querremos una imagen como nuestro componente heroe, para ello hemos creado esta variante.

Heroe dos columnas

Usaremos el atributo variant para elegir el tipo de heroe "two-columns".

Consta de dos columnas personalizables. Para identificar la columna en la que colocaremos el contenido utilizaremos el atributo slot="first-column" o slot="second-column".

Palace es un proyecto de componentes web con código libre. Podrás acceder al código a través de nuestro Github.

Heroe predeterminado

Usaremos el atributo variant para elegir el tipo de heroe "default".

Esta variante corresponde a la vertiente más usada del componente heroe. Consta de una imagen de fondo, un título y un subtítulo.

En el caso de querer personalizar el contenido, se podría realizar mediante el uso de otros elementos dentro de este.
