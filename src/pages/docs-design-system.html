Spinner

<plz-spinner> | PlzSpinner

Los componentes de tipo spinner se utilizan principalmente para indicar que algo está cargando/procesando algún tipo de información. Se suele utilizar haciendo visible el componente spinner al hacer una petición que puede demorar algo de tiempo y cuando se recibe la petición con la información, se vuelve a hacer invisible el componente, de esta forma se indica que algo está sucediendo en ese tiempo de espera.

Ejemplos

Distintos tipos de spinner

Usaremos el atributo variant para elegir la variante de spinner que consideremos entre "simple", "normal", "wave", "atom", "bola".

Considerando la gran variedad de spinners que puedes encontrar en sitios web, se puede decir que estas son las variantes más utilizadas. En cada sitio web estará desarrollado de una manera distinta, en este caso se ha desarrollado intentando mantener el minimalismo, pero a la vez manteniendo su atractivo visual.
