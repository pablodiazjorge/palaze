Footer

<plz-footer> | PlzFooter

El componentefooterse utiliza para mostrar información en forma de sección al final de la página. En el footer suele aparecer información acerca de colaboradores, redes sociales, opciones de contacto, politicas de privacidad y términos de uso, enlaces a páginas relevantes...

Ejemplos

Footer simple

Usaremos el atributo variant para elegir el tipo de footer "simple".

Esta variante es un footer básico con una fina línea en la parte superior para separar el contenido del footer del contenido principal de la página.

Footer con imagen en bloque

Usaremos el atributo variant para elegir el tipo de footer "block".

Esta variante consta de un bloque central personalizable. También se podrá elegir el fondo del footer pasando por el atributo image la url de una imagen.

Footer con imagen con dos columnas

Usaremos el atributo variant para elegir el tipo de footer "two-columns".

Esta variante consta de dos columnas que serán personalizables.

Para identificar la columna en la que colocaremos el contenido utilizaremos el atributo slot="first-column" o slot="second-column" como se muestra a continuación.
